# Contributing to iOS Toast

First off, thank you for considering contributing to iOS Toast! 

## 1. Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](https://github.com/oscarfor10/ios-toast/issues) to see if someone else has already created a ticket. If not, go ahead and create one!

## 2. Fork & create a branch

If this is something you think you can fix, then fork iOS Toast and create a branch with a descriptive name.

## 3. Implement your fix or feature

Make sure your code follows the existing style and conventions.
Since this library aims to have **Zero Dependencies**, please do not add external packages (like jQuery, Lodash, FontAwesome, etc.).

## 4. Make a Pull Request

Submit a Pull Request targeting the `main` branch. Provide a clear description of what you've done. We will review it as soon as possible!
